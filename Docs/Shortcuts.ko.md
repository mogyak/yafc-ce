# YAFC 키보드 단축키

[English](Shortcuts.md) | [한국어](Shortcuts.ko.md)

YAFC는 대부분의 단축키에 플랫폼 기본 보조 키를 사용합니다. Windows와 Linux에서는 `Ctrl`, macOS에서는 `Cmd`입니다.

## 전역 단축키

* **Primary+S** — 프로젝트 변경 사항 저장.
* **Primary+F** — 현재 페이지에서 찾기.
* **Primary+Shift+F** — 페이지 목록에서 찾기.
* **Primary+Z** — 마지막 변경 실행 취소.
* **Primary+Y** 또는 **Primary+Shift+Z** — 실행 취소 다시 실행.
* **Esc** — 현재 검색 닫기.

## 탐색 단축키

* **Primary+N** — NEIE(Never Enough Items Explorer) 열기.

## 탭 단축키

* **Primary+T** — 새 생산 시트, 즉 새 탭 만들기.
* **Primary+Tab** 또는 **Primary+PgDown** — 다음 탭으로 전환.
* **Primary+Shift+Tab** 또는 **Primary+PgUp** — 이전 탭으로 전환.
* **Primary+Shift+PgDown** — 현재 탭을 오른쪽으로 이동.
* **Primary+Shift+PgUp** — 현재 탭을 왼쪽으로 이동.

## 텍스트 입력 단축키

* **Primary+A** — 포커스된 텍스트 상자의 전체 텍스트 선택.
* **Primary+C** — 선택한 텍스트 복사.
* **Primary+V** — 텍스트 붙여넣기.
* **Primary+X** — 선택한 텍스트 잘라내기.
* **Primary+Z** — 포커스된 텍스트 상자의 텍스트 편집 실행 취소.
* **Primary+Backspace** — 이전 단어 삭제.
