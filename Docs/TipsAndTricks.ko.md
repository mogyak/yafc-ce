# YAFC 팁과 요령

[English](TipsAndTricks.md) | [한국어](TipsAndTricks.ko.md)

팁은 게임 진행 순서에 맞춰 정렬되어 있습니다. 전체 목록은 `Yafc/Data/locale/ko/Tips.txt`를 참고하세요.

### 전력 생산

특정 양의 연료가 얼마나 많은 전력을 생산하는지 계산하려고 할 때, 해당 연료를 전력으로 바꾸는 소비 레시피가 목록에 보이지 않을 수 있습니다.
전력 레시피는 특수 레시피이기 때문입니다. 전력 아이콘을 클릭한 뒤 유일한 생산 레시피를 선택하거나, 왼쪽 위 드롭다운 메뉴에서 다음 항목을 선택해 추가해야 합니다.
Recipe > Add Recipe > Electricity.

### 원하는 소비량

때로는 최종 산출물의 양보다 광석 소비량을 기준으로 시트를 구성하는 편이 더 편합니다. 광석을 원하는 산출물에 넣고 음수 값을 지정하면 됩니다.
