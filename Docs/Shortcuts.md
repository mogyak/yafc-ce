# YAFC keyboard shortcuts

[English](Shortcuts.md) | [한국어](Shortcuts.ko.md)

YAFC uses the platform primary modifier for most shortcuts: `Ctrl` on Windows and Linux, `Cmd` on macOS.

## Global shortcuts

* **Primary+S** — Save changes in the project.
* **Primary+F** — Find on the current page.
* **Primary+Shift+F** — Find in the page list.
* **Primary+Z** — Undo the last change.
* **Primary+Y** or **Primary+Shift+Z** — Redo the last undo.
* **Esc** — Close the current search.

## Navigation shortcuts

* **Primary+N** — Show NEIE (the Never Enough Items Explorer).

## Tabs shortcuts

* **Primary+T** — New production sheet (new tab).
* **Primary+Tab** or **Primary+PgDown** — Switch to the next tab.
* **Primary+Shift+Tab** or **Primary+PgUp** — Switch to the previous tab.
* **Primary+Shift+PgDown** — Move the current tab right.
* **Primary+Shift+PgUp** — Move the current tab left.

## Text input shortcuts

* **Primary+A** — Select all text in the focused text box.
* **Primary+C** — Copy selected text.
* **Primary+V** — Paste text.
* **Primary+X** — Cut selected text.
* **Primary+Z** — Undo text edits in the focused text box.
* **Primary+Backspace** — Delete the previous word.
